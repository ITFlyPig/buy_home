# 需求分析

## 用户画像
- 在杭州工作的购房刚需/改善群体
- 关注学区、通勤、配套、预算等多维度信息
- 需要数据支撑辅助决策，而非纯主观推荐

## 功能需求

### P0 - 核心功能

#### 1. 板块总览地图
- 杭州各区/板块房价热力图
- 点击板块查看详情（均价、在售楼盘、学校等）
- 支持按预算范围筛选高亮

#### 2. 学区匹配查询
- 数据来源：浙里办「入学早知道」
- 输入小区名 → 查看对口学校
- 输入学校名 → 查看对口小区及房价
- 学校评级标注（省重点、市重点等）

#### 3. 板块评分对比
- 多维度雷达图对比（最多选3个板块）
- 评分维度：房价、学区、通勤、配套、潜力、品质
- 输出综合推荐指数

#### 4. 价格趋势分析
- 近1年各板块二手房成交均价走势
- 新房开盘价格分布
- 标注政策节点（如限购松绑时间）

### P1 - 增强功能

#### 5. 通勤计算器
- 输入工作地点，计算各板块通勤时间
- 支持公交/地铁/自驾多种方式
- 在地图上标注通勤圈（30min/45min/60min）

#### 6. 购房成本计算器
- 输入总价/首付比例/贷款年限
- 计算月供、总利息、税费
- 公积金+商贷组合计算

#### 7. 预算推荐
- 输入预算范围 + 需求偏好（学区优先/通勤优先等）
- 智能推荐匹配板块
- 列出该预算段热门楼盘/小区

### P2 - 扩展功能

#### 8. 新盘日历
- 即将开盘的新盘信息
- 摇号中签率预测

#### 9. 政策速查
- 当前限购政策一览
- 落户要求与购房资格

## 非功能需求

- 数据每周更新
- 移动端适配（H5 版本）
- 页面加载 < 3s
