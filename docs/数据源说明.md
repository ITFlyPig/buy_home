## 一、浙里办 - 学区数据（核心数据源）

### 数据来源
- **平台**: 杭州教育局「入学早知道」系统
- **官方网站**: https://rxyj.hzedu.gov.cn
- **数据内容**: 学校基础信息、学区划分、学校-小区对应关系

### 数据采集方式
通过「入学早知道」API接口采集，相比浙里办APP更加稳定高效：
- 接口地址: https://rxyj.hzedu.gov.cn/api/...
- 无需登录，公开可访问
- 数据更新及时，与官方同步

## 二、贝壳找房 - 小区价格数据

### 数据来源
- **平台**: 贝壳找房杭州站
- **官方网站**: https://hz.ke.com/xiaoqu/
- **数据内容**: 小区均价、总价范围、户型信息

### 数据采集方式
通过双接口重试机制采集：
1. **PC端搜索接口**: https://hz.ke.com/xiaoqu/rs{小区名}/
2. **移动端接口**: https://m.ke.com/search/result?q={小区名}&city=hz

### 核心字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| name | string | 小区名称 | 友谊新村 |
| avg_price | int | 均价（元/㎡） | 52000 |
| min_total | int | 最低总价（万） | 350 |
| max_total | int | 最高总价（万） | 700 |
| layout | string | 主力户型 | 2室(65-85㎡) |
| schools | array | 对口学校 | ["杭州市文三路小学"] |
| districts | array | 所属区域 | ["西湖区"] |
| data_source | string | 数据来源 | ke.com |

### 数据文件

| 文件 | 格式 | 记录数 | 说明 |
|------|------|--------|------|
| data/processed/community_prices.json | JSON | 9712 | 小区价格主数据 |
| visualization/data/community_prices.js | JS | 9712 | 前端数据文件 |

### 采集脚本使用

```bash
# 全量采集（约 9700+ 个小区，预计8-10小时）
python3 data/scripts/crawl_community_prices.py

# 按区域采集
python3 data/scripts/crawl_community_prices.py --district 西湖区
python3 data/scripts/crawl_community_prices.py --district 上城区

# 限制采集数量（测试用）
python3 data/scripts/crawl_community_prices.py --limit 50
```

### 采集特性
- **断点续爬**: 中断后重新运行自动继续
- **进度保存**: 每10条数据自动保存一次
- **双接口重试**: PC端失败自动尝试移动端
- **反爬策略**: 随机User-Agent，3-5秒请求间隔

## 三、手动补充数据

### 数据来源
部分学校详细信息无法通过API获取，从以下权威网站手动补充：
- **学校官网**: 官方发布的学校简介、师资力量、办学特色
- **教育部门网站**: 杭州市教育局、各区教育局公告
- **权威媒体**: 升学在线、学区房网等专业教育媒体

### 补充内容
- 学校详细地址
- 办学规模（班级数、学生数、教师数）
- 中考成绩（重点高中升学率）
- 学校特色和荣誉

### 数据文件
- `data/zheliban/school_detail.json` - 学校详细信息

## 六、数据质量说明

### 数据优先级
1. **原始数据优先**: `data/raw/` 和 `data/zheliban/` 目录中的数据为权威来源
2. **自动采集**: 小区价格数据通过爬虫采集，准确性依赖数据源
3. **手动补充**: 学校详情数据从权威网站手动录入

### 注意事项
- 学区划分每年可能微调，需关注最新公告
- 部分学校有多校划片政策
- 九年一贯制学校需特别标注
- 小区价格为挂牌均价，非实际成交价
