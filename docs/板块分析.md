## 板块评分模型（待数据填充）

评分标准（1-10分）：

| 维度 | 评分依据 |
|------|----------|
| 房价 | 均价越低分越高（对刚需友好） |
| 学区 | 省重点+2分，市重点+1分 |
| 通勤 | 到钱江新城CBD地铁通勤时间 |
| 配套 | 1km内地铁+商场+医院数量 |
| 潜力 | 规划利好+人口流入+价格涨幅 |
| 品质 | 次新房占比+物业水平+绿化率 |

## 预算分段推荐

### 200万以下
- 临平乔司、钱塘下沙、余杭闲林
- 特点：面积小/老小区，通勤时间较长

### 200-300万
- 临平新城、萧山城区、余杭仓前、丁桥
- 特点：刚需主力段，可选择较多

### 300-500万
- 未来科技城、三墩、祥符、市北、浦沿
- 特点：品质刚需/刚改，配套逐步成熟

### 500-800万
- 申花、蒋村、西兴、长河、四季青
- 特点：改善型，学区+配套兼顾

### 800万以上
- 钱江新城、文教区、桥西豪宅
- 特点：顶级学区或核心地段

> ⚠️ 以上价格为参考范围，实际需结合具体楼盘和面积段
